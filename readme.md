## Зависимости
    php: 7.1<
    node.js: 8.4

## Команды установки
    composer install
    npm install

## Команды для базы данных
    php bin/console doctrine:database:create
    php bin/console doctrine:migrations:migrate

## Сборка ассетов
    npm run dev

## Запуск сервера
    php bin/console server:start
